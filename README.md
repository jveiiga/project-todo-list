## Instalação

    - Clonar repositorio
    $ git clone https://github.com/jveiiga/project-todo-list.git

    FRONT-END 📺
    
    $ cd project-todo-list

    - Instalação de dependências
    $ yarn install
    ou
    $ npm install

    - Rodar projeto
    $ yarn dev
    ou
    $ npm dev
